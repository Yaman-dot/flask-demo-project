# Flask Demo Project

This is a **Flask test/demo project** designed to showcase:

- Handling form inputs and POST requests
- Integrating a machine learning model for predictions
- Multi-page routing (`/`, `/about`, `/donut`, `/filters`, etc.)
- HTML/CSS/JS front-end integration
- Displaying results dynamically

> The heart disease predictor is included as an example ML model to demonstrate functionality.
